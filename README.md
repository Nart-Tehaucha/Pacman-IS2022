## Pacman-IS2022

### Project Description:
PacMan game created with Java - Software Development course @ University of Haifa.

First place winner!

---

* Database management
* Game logic - Game states, Navigation, basic AI, Map creation.
* GUI combining Swing and JavaFX.
* Unit testing through JUnit.
